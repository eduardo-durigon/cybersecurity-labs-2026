## 02-linux-investigation-lab

In this lab, I practiced beginner Linux skills in a cybersecurity-focused scenario on an Ubuntu Server VM using UTM on a Mac M2.

### Skills practiced
- Navigating directories with `pwd`, `ls`, and `cd`
- Creating folders with `mkdir`
- Creating files with `touch`
- Reading file contents with `cat`
- Changing file permissions with `chmod`

### Scenario
I acted as a junior cybersecurity analyst investigating suspicious activity on a Linux server. I created investigation folders, reviewed simulated log files, inspected a suspicious shell script, and adjusted file permissions to protect sensitive files.

### Outcome
This lab helped me reinforce foundational Linux commands while applying them to a more realistic security investigation workflow.
